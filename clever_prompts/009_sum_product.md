# rolling_max

## Signature

```rust
pub fn rolling_max(numbers: &[i64]) -> Vec<i64>
```

## Docstring

From a given list of integers, generate a list of rolling maximum element found until given moment
in the sequence.

(Note: CLEVER's row for this problem has `sum_product(...) -> (int, int)` as the
signature, but the docstring describes rolling-max. Our Rust crate follows the
docstring; the signature here is renamed and re-typed to match.)
